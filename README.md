# Abhijeet Yadav Portfolio

## Overview

This is my personal developer portfolio built to showcase my skills, projects, technical interests, and contact information. The portfolio serves as a living portfolio that grows with my learning journey. Instead of remaining static, it will be updated regularly as I build new projects and improve my skills.

The website is designed with a modern UI, smooth animations, responsive layouts, dark/light mode support, and an integrated contact form that allows visitors to send messages directly to my email.

---

* Showcase my technical skills.
* Present my projects professionally.
* Provide recruiters with an easy way to contact me.

# How to Run Locally

Clone the repository:

```bash
git clone https://github.com/abhijeetyadav37/portfolio.git
```

Move into the project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
